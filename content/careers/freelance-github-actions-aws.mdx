As a **side freelance engagement** I took a **Next.js** project and brought it to a **fully automated** lifecycle: **GitHub Actions** for CI (build, test, code quality), **automatic deployment**, and **infrastructure provisioning** on **AWS** (EC2, IAM, and related services).

**What I delivered:**

- **Continuous integration:** Workflows for build, test, and code quality checks on every push or PR.
- **Deployment automation:** From merge to deployment with minimal manual steps.
- **Infrastructure as Code:** Provisioning of EC2, networking, and IAM via **Terraform** (or equivalent) so the environment was versioned and repeatable.
- **Documentation:** Clear runbooks and pipeline descriptions so the client could maintain or extend the setup.
