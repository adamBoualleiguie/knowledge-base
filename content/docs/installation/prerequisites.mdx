# Prerequisites

Before installing, make sure you have the following:

## Required Software

- **Node.js**: Version 20.19.4 or higher
- **Yarn**: Package manager
- **Git**: Version control

## System Requirements

- Operating System: macOS, Linux, or Windows
- RAM: Minimum 4GB
- Disk Space: At least 500MB free

## Verification

Check your versions:

```bash
node --version
yarn --version
git --version
```

Once you have these installed, proceed to the [Setup Guide](/docs/installation/setup).
