# Deployment Overview

This section covers how to deploy your portfolio website to various platforms.

## Deployment Options

- **Vercel** (Recommended) - Optimized for Next.js
- **Netlify** - Easy deployment with continuous integration
- **Static Export** - Deploy as static files

## Quick Links

- [Deploy to Vercel](/docs/guide/deployment/vercel) - Step-by-step Vercel deployment
- [Deploy to Netlify](/docs/guide/deployment/netlify) - Step-by-step Netlify deployment
- [Static Export](/docs/guide/deployment/static-export) - Deploy as static files

## Prerequisites

Before deploying:

1. **Build your site**: `yarn build`
2. **Test locally**: `yarn start`
3. **Push to Git**: Commit and push to GitHub/GitLab
