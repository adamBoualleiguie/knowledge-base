# API Endpoints

Documentation for all available API endpoints.

## Authentication

### POST /api/auth/login

Authenticate a user.

**Request Body:**
```json
{
  "email": "user@example.com",
  "password": "password123"
}
```

**Response:**
```json
{
  "token": "jwt-token-here",
  "user": {
    "id": "123",
    "email": "user@example.com"
  }
}
```

## Content

### GET /api/docs

Get all documentation pages.

**Response:**
```json
{
  "docs": [
    {
      "title": "Getting Started",
      "url": "/docs/getting-started",
      "description": "..."
    }
  ]
}
```
